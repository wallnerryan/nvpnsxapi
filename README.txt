
Nicira NVP/NSX Python API
=========================

*Note, all api calls have been tested on NVP 3.0-3.2
*Note, NOT all api calls have been testing on NSX, but most will still work

About
=========================

This API library is used to do 2 things.
	1. Setup Hypervisor Transport, Gateway, and Service Nodes
	2. Provide Python Bindings for Nicira's NVP/NSX Platform APIs.
